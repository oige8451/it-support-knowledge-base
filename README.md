# IT Support Knowledge Base

A collection of Tier 1 help desk SOPs and troubleshooting guides 
covering the most common issues in enterprise IT support environments.

## Purpose

This knowledge base documents step-by-step resolution procedures 
for common end-user issues, following help desk best practices for 
clear, repeatable troubleshooting workflows.

## How to use this knowledge base

Each guide follows the same format:
- **Issue** — what the user reports
- **Possible causes** — what might be causing it
- **Step-by-step resolution** — exact steps to fix it
- **Escalation criteria** — when to escalate to Tier 2
